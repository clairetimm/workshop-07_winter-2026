# workshop-07_winter-2026
Week 7 workshop: linear models

## General information

This repository contains two examples for running linear models.

This is my personal repo forked from the ENVS-193DS-2026 organization. I am the sole contributor. 

## Data and file information

All code files are in the 'code' folder. These are .qmd files that contain models for abalone-pH ('abalone-example.qmd) and temperature ('sonadora-temperature-example.qmd). 

Both files include code for creating linear models, checking model diagnostics, creating predictions, and visualizing predictions.
